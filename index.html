Pascallove2d

A source to source compiler allowing Pascal developers to reuse code and develop games for the Love2D framework.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to GitHub Pages.

Welcome to the project page. This project aims to develop a pascal compiler to allow source-source compilation for pascal-LUA with a particular focus of providing a functional output that implements an interface over the Love2D framework (https://love2d.org/).
